Частые вопросы
Насколько удобен для “чайника” ваш личный кабинет?
Мы можем предоставить демо-доступ к тестовому личному кабинету, чтобы вы могли его оценить. И наш менеджер вас по нему проконсультирует. Для подачи заявки, позвоните нам по телефону +7 (495) 137-56-66
Для использования облачной кассы оплаты только картами должны производиться или с наличными тоже можно работать?
Сервис Атол Онлайн позволяет фискализировать наличные на облачной кассе для осуществления следующих видов расчетов:
  • В салоне транспортного средства (при реализации проездных билетов).
  • Оплата товара курьеру или продавцу при разносной торговле.
  • Оплата выполненных работ или оказанных услуг вне торгового объекта.
Подходит ли сервис АТОЛ Онлайн для организаций, у которых будет удержание денег из зарплаты сотрудникам?
Да, сервис АТОЛ Онлайн можно использовать для фискализации удержания денежных средств из заработной платы сотрудников. По закону, работодатель обязан применять онлайн-кассу, если из заработной платы работника удерживается стоимость:
  • реализуемых ему товаров (см. понятие «расчет», приведенное в ст. 1.1 Закона № 54-ФЗ, разъяснения в письме Минфина России от 30.11.2018 № 03-01-15/86884);
  • оказанных ему услуг (см. п. 4 ст. 4 Закона № 192-ФЗ, письмо Минфина России от 25.01.2019 № 03-01-15/4355);
  • выданной форменной одежды (письмо Минфина России от 12.12.2018 № 03-01-15/90372).
Требуется ли вручную закрывать и открывать смены на кассе?
Нет. Открытие и закрытие смен на онлайн-кассах происходит автоматически в 02 00 по мск. Время открытия и закрытия смен может быть скорректировано по запросу, напишите в техподдержку.
Как разработать прямую интеграцию и произвести тестирование?
  1. Протокол API доступен по ссылке.
  2. Параметры тестовой среды здесь - Общедоступный тестовый контур. Используется на этапе разработки, для тестов на отработку логики интеграции.
  3. Тестовый Личный Кабинет доступен по ссылке - Для подключения выделенной тестовой ККТ .Обычно, используется после завершения разработки интеграции. Для тестов «под нагрузку», в целях определения кол-ва ККТ.
Что делать если кассы больше не нужны?
Рекомендуем расторгнуть договор.
  1. Уведомите нас через оператора контактного центра или отправьте запрос на электронную почту 1@atol.ru.
  2. Снимите кассу с учета в налоговой.
  3. Заберите свой фискальный накопитель или заключите с нами договор на хранение. Стоимость хранения 1 устройства 12 месяцев – 600 руб.
Сотрудники контактного центра готовы проконсультировать по любому вопросу, связанному с расторжением договора.
Сколько мне нужно касс?
  1. Если количество транзакции минимальное, возьмите 2 кассы, чтобы обеспечить бесперебойную работу интернет-магазина.
  2. Посчитайте оптимальное количество при помощи онлайн-калькулятора АТОЛ-Онлайн или на 3-ем шаге регистрации в личном кабинете сервиса.
  3. Определите количество касс самостоятельно, исходя из нагрузки на 1 кассу. 1 касса на срок от 15 месяцев может записать до 250 000 транзакции за весь период эксплуатации. Пропускная способность – 1 чек в секунду
Как поставить кассу на учет?
  1. Перейдите на сайт Федеральной налоговой службы.
  2. Войдите в личный кабинет или зарегистрируйтесь.
  3. Перейдите в раздел «Учет ККТ».
  4. Нажмите зарегистрировать ККТ.
Или заполните форму по КНД № 1110061 на сайте Федеральной налоговой службы и подайте заявление на регистрацию кассы лично.
Как настраивать интеграцию?
АТОЛ Онлайн поддерживает входящую интеграцию, которая может быть:
  1. Прямой - реализованная силами и средствами клиента. Протокол API доступен по ссылке;
  2. Готовой - реализованная через Партнера АТОЛ Онлайн. Список партнёров доступен по ссылке.

По завершении всех шагов регистрации, в разделе личного кабинета АТОЛ Онлайн «Мои компании» автоматически формируется файл настроек для завершения интеграции, который содержит данные о компании логин, пароль и код группы КТТ - (login/password/group_code).

Для получения этих данных необходимо нажать клавишу «Настройки Интегратора».

Эти параметры требуется передать в платежную систему, банк интернет-эквайринга, CMS или другую учетную систему, с которым у сервиса АТОЛ Онлайн выполнена интеграция.

Когда можно будет пользоваться Кассой после ее регистрации?
После подключения услуг ОФД и настройки интеграции с кассой.
